## 🚀 About Project

TEAM HORCRUX Compressor is a modern Chrome Extension developed for quick and easy file compression directly inside the browser.

It provides:

✅ Lossy Compression  
✅ Lossless Compression  
✅ Download Compressed File  
✅ Progress Bar  
✅ Professional UI
